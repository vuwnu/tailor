what is this?

Tailor is a classless CSS framework, designed for easily giving style to a HTML document that has none

how to use?

the framework can either be downloaded and installed locally or linked to through a cdn

css variables

Tailor has two different versions. One of these versions makes use of CSS custom variables to give easy control over certain parts of how your site looks
